# 🐍🎲 Snakes and Ladders

A fun and interactive implementation of the classic **Snakes and Ladders** board game built using **HTML**, **CSS**, and **JavaScript**. Players roll the dice, navigate the board, climb ladders, avoid snakes, and race to reach the final square first.

## 📖 Overview

Snakes and Ladders is one of the most popular board games worldwide. This web-based version recreates the traditional gameplay experience with an intuitive user interface, dynamic animations, and responsive design.

The project demonstrates JavaScript game logic, event handling, DOM manipulation, and front-end development techniques while providing an enjoyable gaming experience.

## ✨ Features

* Interactive game board
* Random dice rolling functionality
* Player movement across the board
* Snake and ladder mechanics
* Automatic position updates
* Win detection system
* Responsive design for desktop and mobile devices
* Simple and user-friendly interface
* Lightweight implementation with no external dependencies

## 🛠️ Technologies Used

* JavaScript (ES6)

## ▶️ How to Play

1. Start the game.
2. Click the dice button to roll.
3. Move your token according to the dice value.
4. Land on a ladder to climb upward.
5. Land on a snake head and slide downward.
6. Continue rolling until a player reaches the final square.
7. The first player to reach the last square wins the game.

## 🎮 Game Rules

* Players take turns rolling the dice.
* The dice value determines movement.
* Ladders help players advance faster.
* Snakes move players backward.
* Exact or valid finishing rules may apply depending on implementation.
* The first player to reach the destination wins.
